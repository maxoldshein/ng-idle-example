Ng-Idle Example

This is an example of how to utilize the ng-idle framework. In this example, we are allowing the user 5 minutes of total idleness, with a 'warning' that lasts for 2 minutes. After 3 minutes of no interruption events occuring, the user will then become 'IDLE'. I have included the below indicators in an attempt to provide a visualization of what is going on behind the scenes within the code itself.

You are currently considered:
NOT IDLE

You are currently considered:
IDLE

Last interruption occured {{idleMinutes}} minute(s) and {{idleSeconds % 60}} second(s) ago.

The last interruption event that occured was:
{{lastEvent}}
